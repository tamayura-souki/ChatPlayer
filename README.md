# ChatPlayer
youtubeチャットで遊ぶためのコード。チャットを取得して、コマンドを認識して、GBで表示したりするアプリ、配信画面に乗せて使う。

デフォルトでコメントがウインドウ上に右から左に流れるようになっている。

## 実装済みのコマンド
/police /medic サイレンを鳴らす
/w 笑い声を鳴らす

/red /blue /green /bk 文字色を変更する
/fast /slow 文字の流れる速度を変える
/-fast /-run /-slow 文字を逆から流す

/unk チャンネル名非表示

# Requirements
- selenium(chrome)
- pygame
- emoji(emoji削除用)
